# Repo Structure

This repo hosts the Flux marketing site, editor landing page, and shared brand utilities.

## Key Directories

- `src/` — Marketing site pages, docs shell, and shared UI components.
- `packages/brand/` — Shared brand identity renderers and helpers.
- `packages/viewer/` — Viewer and editor build artifacts.
- `scripts/` — Build helpers such as version synchronization and 404 copy scripts.

## Supporting Files

- `vite.config.ts`, `tailwind.config.js`, and `postcss.config.js` configure the build pipeline.
- `tests/` holds Vitest and Playwright test coverage.
